Fake Review Detection Using Machine Learning

A machine learning model to classify reviews as genuine or fake using Natural Language Processing (NLP) techniques.

 Features
- Text preprocessing: tokenization, stopword removal, stemming  
- Feature extraction: TF-IDF and Bag-of-Words  
- Model training using Logistic Regression and SVM  
- Achieved **94% accuracy** on test data  

 Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib
